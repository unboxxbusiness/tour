# **App Name**: Next Starter

## Core Features:

- Home Page Layout: Create a clean layout with a simple, Tailwind-styled home page.
- Tailwind Configuration: Configure Tailwind CSS for styling the application.
- Global Font and Reset: Add a global font and basic CSS reset to ensure consistent styling across the app.

## Style Guidelines:

- Primary color: Deep indigo (#3F51B5) to evoke a sense of trust and stability in the digital space.
- Background color: Light gray (#F5F5F5), very desaturated indigo, to ensure a clean, uncluttered backdrop that keeps the focus on the content.
- Accent color: Amber (#FFC107), a vivid analogous to the primary to bring warmth and signals interaction points to the users attention.
- Body and headline font: 'Inter' sans-serif for a modern and clean look, providing excellent readability.
- Use a simple, full-width layout with a central container for content. Max width should be 1200px.
